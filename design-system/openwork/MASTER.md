# OpenWork 视觉系统

## 核心气质

OpenWork 是一张安静的“全球工作地图册”，不是传统招聘网站。界面应当像一本正在展开的编辑部地图：克制、留白、真实、有一点探索世界的浪漫。

设计原则：

- 地球永远是视觉主角，界面只提供必要线索。
- 首页负责建立情绪，探索页负责直接工作。
- 信息按需出现：默认看分布，悬停看摘要，点击后看完整岗位。
- 动画表现空间变化，不做无意义装饰。
- 桌面、手机网页与小红书离线版保持同一种气质。

## 色彩

| 角色 | 色值 | 用途 |
| --- | --- | --- |
| 画布 | `#F3F0E9` | 全局暖白背景 |
| 纸张 | `#FFFDF8` | 搜索框与浮层 |
| 墨色 | `#12191C` | 标题、主按钮 |
| 次级墨色 | `#49565A` | 正文与辅助信息 |
| 粉雾浅蓝 | `#BBD2DA` | 无渐变、与陆地有清晰层级的纯色海洋 |
| 淡雾青 | `#D7E6E7` | 仅用于地球外缘的薄氛围 |
| 珊瑚橙 | `#FF6B52` | 入口动作与关键坐标 |
| 鼠尾草绿 | `#76B391` | 岗位状态与地图点 |
| 边界线 | `rgba(18,25,28,.12)` | 轻边框与结构分隔 |

国家板块使用高明度、低饱和但不发白的莫兰迪色；青绿色板应偏鼠尾草绿，灰蓝色板应偏灰紫，避免与海洋相融。海洋内部必须保持纯色，不使用径向渐变、镜面高光或明暗蓝色。岗位点可以更鲜明，但避免荧光感。珊瑚橙只用于关键动作或少量坐标，不大面积铺色。

## 字体与层级

- 中文大标题：宋体风格系统字体，强调编辑感与人文气质。
- 英文岗位标题：Iowan Old Style / Palatino 等传统衬线字体。
- 界面文字：系统无衬线字体，简洁、清楚。
- 元信息：等宽字体，小字号、适度字距，像地图编号。
- 首页标题使用极大字号和紧凑行距；其余界面不与标题争夺注意力。

## 布局

### 首页

- 左侧叙事，右侧大地球。
- 顶部只保留品牌和最近 30 天状态。
- 唯一主动作是“探索工作世界”。
- 地球上的岗位点不显示文字，只表现全球分布。

### 探索页

- 沿用首页同一个地球，通过放大和位移完成过渡。
- 初始不显示岗位卡片。
- 搜索框悬浮在底部正中，不显示筛选器。
- 默认只显示彩色坐标点；悬停后才显示光环、数量或公司标识和摘要。
- 点击坐标后，岗位卡片从上方进入。

### 岗位卡片

- 来源固定在右上角。
- 公司、岗位、地点、薪资和摘要按阅读顺序排列。
- “立即投递”和“查看详情”在左下角单独成行。
- 左右切换按钮在卡片外缘；手机端移入卡片底部。

### 手机

- 首页先看标题，地球从下方进入画面。
- 探索页只露出地球局部，搜索框贴近底部安全区。
- 搜索结果控制在屏幕下半部；岗位卡片使用近全宽布局。
- 横屏不产生额外滚动。

## 动效

- 首页到探索页：860ms 空间过渡，同一地球连续放大。
- 微交互：150–300ms，使用自然的缓出曲线。
- 岗位卡片：轻微上浮、淡入与模糊消散。
- 地球缓慢自动旋转；用户操作时暂停，空闲后恢复。
- 尊重 `prefers-reduced-motion`，关闭非必要位移和持续动画。

## 禁止事项

- 不堆放统计数字、图例、筛选器和解释面板。
- 不使用玻璃拟态炫光、霓虹渐变和高饱和科技蓝。
- 不让所有坐标同时显示文字或数字。
- 不使用表情符号充当图标。
- 不隐藏键盘焦点，不制造横向滚动。
- 不为追求动效打断搜索、投递和地图操作。

## 交付检查

- [ ] 首页、探索、搜索、选中岗位四个状态视觉连续。
- [ ] 390×844、844×390、1440×1000 无横向滚动或遮挡。
- [ ] 搜索、结果选择、关闭、前后切换、返回首页可用。
- [ ] 悬停信息清楚，触屏点击区域足够大。
- [ ] 动效减弱模式可用。
- [ ] 小红书离线包无外部依赖且校验通过。
