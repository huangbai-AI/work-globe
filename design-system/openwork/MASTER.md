# OpenWork 视觉系统

## 核心气质

OpenWork 是一张安静的“全球工作地图册”，不是传统招聘网站。界面应当像一本正在展开的编辑部地图：克制、留白、真实、有一点探索世界的浪漫。

设计原则：

- 地球永远是视觉主角，界面只提供必要线索。
- 首页负责建立情绪，探索页负责直接工作。
- 信息按需出现：默认看分布，悬停看摘要，点击后看完整岗位。
- 动画表现空间变化，不做无意义装饰。
- 桌面、手机网页与小红书离线版保持同一种气质。

| 海洋浅蓝 | `#E1ECF7` | 参考图海洋的精确主色 |
| 冷白纸张 | `#FCFCFC` | 搜索框与浮层 |
| 墨色 | `#111417` | 标题、主按钮 |
| 次级墨色 | `#4E5559` | 正文与辅助信息 |

| 坐标橙 | `#F3842D` | 入口动作与少量坐标 |
| 坐标蓝 | `#2181D5` | 键盘焦点与地图点 |
| 坐标绿 | `#48934B` | 岗位状态与地图点 |

国家板块固定使用参考图提取出的浅粉 `#E6CCCB`、鼠尾草 `#CBD6CD`、灰蓝 `#C8D4E6`、砂岩 `#E8D9C4`、淡紫 `#D9CEE1`、雾粉 `#E5D2D3`、灰青 `#CAD6D1`、暖砂 `#E8D3BD`、蓝紫灰 `#C7C8DC` 与藕粉 `#E2CDD0`。国界统一使用接近不透明的纯白线，并配合浅色侧壁形成清晰描边。画布保持 `#E2EDF8`，海洋内部保持 `#E1ECF7` 纯色，不使用海洋渐变、镜面高光或明暗蓝色。图钉固定使用参考图的 14 个实测色号；常规图钉只保留彩色圆头、克制的左上高光、下缘暗部与细针脚，不使用白色外环。

## 字体与层级

- 中文大标题：宋体风格系统字体，强调编辑感与人文气质。
- 英文岗位标题：Iowan Old Style / Palatino 等传统衬线字体。
- 界面文字：系统无衬线字体，简洁、清楚。
- 元信息：等宽字体，小字号、适度字距，像地图编号。
- 首页标题使用极大字号和紧凑行距；其余界面不与标题争夺注意力。

## 布局

### 首页

- 左侧叙事，右侧大地球。
- 顶部只保留品牌，不显示时间状态或辅助标签。
- 唯一主动作是“探索工作世界”。
- 地球上的岗位点不显示文字，只表现全球分布。

### 探索页

- 沿用首页同一个地球，通过放大和位移完成过渡。
- 初始不显示岗位卡片。
- 搜索框悬浮在底部正中，不显示筛选器。
- 默认只显示彩色坐标点；悬停后才显示公司标识和摘要，不额外增加白色光环。
- 点击坐标后，岗位卡片从上方进入。

### 岗位卡片

- 来源固定在右上角。
- 公司、岗位、地点、薪资和摘要按阅读顺序排列。
- “立即投递”和“查看详情”在左下角单独成行。
- 左右切换按钮在卡片外缘；手机端移入卡片底部。

### 手机

- 首页先看标题，地球从下方进入画面。
- 探索页只露出地球局部，搜索框贴近底部安全区。
- 搜索结果控制在屏幕下半部；岗位卡片使用近全宽布局。
- 横屏不产生额外滚动。

## 动效

- 首页到探索页：860ms 空间过渡，同一地球连续放大。
- 微交互：150–300ms，使用自然的缓出曲线。
- 岗位卡片：轻微上浮、淡入与模糊消散。
- 地球缓慢自动旋转；用户操作时暂停，空闲后恢复。
- 尊重 `prefers-reduced-motion`，关闭非必要位移和持续动画。

## 禁止事项

- 不堆放统计数字、图例、筛选器和解释面板。
- 不使用玻璃拟态炫光、霓虹渐变和高饱和科技蓝。
- 不让所有坐标同时显示文字或数字。
- 不使用表情符号充当图标。
- 不隐藏键盘焦点，不制造横向滚动。
- 不为追求动效打断搜索、投递和地图操作。

## 交付检查

- [ ] 首页、探索、搜索、选中岗位四个状态视觉连续。
- [ ] 390×844、844×390、1440×1000 无横向滚动或遮挡。
- [ ] 搜索、结果选择、关闭、前后切换、返回首页可用。
- [ ] 悬停信息清楚，触屏点击区域足够大。
- [ ] 动效减弱模式可用。
- [ ] 小红书离线包无外部依赖且校验通过。
