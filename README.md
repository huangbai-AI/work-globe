# 工作候鸟

一个从公开招聘帖里发现全球工作机会的 3D 地球网站。

## 连续场景

- `index.html`：首页与搜索地图共用同一个场景、同一颗地球。
- 点击“开始查看”后，地球从右侧平滑移到主视野，首页文案退场，搜索栏从底部出现；过程中不跳页、不重建地球。
- 搜索状态采用“上方岗位浮卡、下方局部地球、底部单行搜索条”的构图；浮卡可左右切换岗位，并带动地球转向对应地点。
- `explore.html`：仅用于兼容旧链接，会把原有查询条件带回连续场景。

## 数据

当前原型收录 1,464 个最近 30 天的公开招聘线索，其中 902 个为明确远程岗位。来源包含 X、Reddit、领英、AnySearch 全球职位搜索、Remote OK、Remotive、Arbeitnow、We Work Remotely、Himalayas、Jobicy、NoDesk 与公开职位页。每条数据都保留原帖或职位页链接。

其中 738 个岗位能落到 254 座具体城市。地图只接受两类坐标：职位正文明确写出的城市，或招聘来源中能核实的公司总部城市。只写国家、地区、时区或“全球远程”的岗位仍可搜索和浏览，但不会再用国家中心或海上坐标冒充城市点。

远程岗位不限定方向，除设计、开发、人工智能外，还覆盖客服、行政、财务、教育、医疗、写作、法务、销售、市场和日常运营。

- 光点颜色：岗位类型
- 光点大小：公开互动关注度
- 光点亮度：当前搜索是否匹配
- 光点位置：职位城市；若使用公司总部，悬停信息会明确标注

远程职位可通过底部的“全球远程”选项一键筛选。

城市坐标可重新生成：

```bash
node scripts/enrich_locations.mjs
```

## 本地运行

```bash
python3 -m http.server 4182
```

打开 `http://127.0.0.1:4182`。
